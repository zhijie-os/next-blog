## Split vertically 
```
Ctrl-b %
```


## Split horizontally 
```
Ctrl-b "
```

## Move focus to pane
```
Ctrl-b <arrow-key>
```

## Create a new window
```
Ctrl-b c
```


## Renaming current window
```
Ctrl-b ,
```

## Go to next window
```
Ctrl-b n
```

## Go to prev window
```
Ctrl-b p
```

## Go to window index

```
Ctrl-b 0-9
```
